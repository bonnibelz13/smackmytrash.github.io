SMACK MY TRASH

"Smack My Trash" เป็นโครงงานที่เน้นการจัดการขยะ
เป้าหมายหลักของโครงงานนี้คือการพัฒนาถังขยะให้สามารถเปิดปิดอัตโนมัติด้วยการใช้เซนเซอร์อัลตราโซนิก (Ultrasonic Sensor) และแสดงข้อมูลผ่านจอ LCD ที่เชื่อมต่อกับไมโครคอนโทรลเลอร์บอร์ด Arduino Uno R3

ถังขยะอัจฉริยะ "SMACK MY TRASH" เป็นโครงงานที่จัดทำขึ้นเพื่ออำนวยความสะดวก
ในการทิ้งและจัดการขยะ โดยพัฒนาถังขยะให้สามารถเปิด-ปิดอัตโนมัติได้จากการตรวจสอบระยะของวัตถุด้วย Ultrasonic Sensor และสามารถแจ้งเตือนสถานะได้เมื่อถังขยะเต็ม โดยแสดงข้อมูลผ่านจอภาพผลึกเหลว (Liquid Crystal Display: LCD) และไดโอดเปล่งแสง (LED) ซึ่งการทำงานทั้งหมดถูกควบคุมด้วยบอร์ดไมโครคอนโทรลเลอร์ Arduino Uno R3 ที่ประกอบด้วยชุดคำสั่งภาษา C ที่ผู้จัดทำได้พัฒนาขึ้น

Library & void setup

เรียกใช้ไลบรารีและกำหนดค่าเริ่มต้น

เรียกใช้ไลบรารี Wire.h, LiquidCrystal_I2C.h, และ Servo.h
ซึ่งจำเป็นสำหรับการใช้งาน I2C LCD และ Servo Motor.

        ประกาศตัวแปร myservo เพื่อใช้ในการควบคุม Servo Motor และ lcd เพื่อใช้งานหน้าจอ LCD และประกาศตัวแปรสำหรับขาของ Arduino และสถานะ state และสี LED (red, green, blue)

    ในฟังก์ชัน setup() เรียก Serial.begin(9600) เพื่อเปิดการสื่อสารผ่าน Serial ที่อัตราบิต 9600 และเรียก lcd.init() เพื่อเริ่มต้นหน้าจอ LCD และ lcd.backlight() เพื่อเปิดไฟหน้าจอ LCD กำหนดขาของ Arduino ที่ใช้ในการควบคุม Servo และระบบ LED โดยใช้ pinMode() และกำหนดตำแหน่งข้อความเริ่มต้นบนหน้าจอ LCD



void open

ฟังก์ชันสำหรับเปิดถัง

เรียกใช้ digitalWrite() เพื่อตั้งสี LED เป็นสีเขียว เพื่อบ่งชี้สถานะการเปิดถังและเรียก lcd.clear() เพื่อลบข้อมูลบนหน้าจอ LCD และ lcd.setCursor() และ lcd.print() เพื่อแสดงข้อความ "OPEN" ควบคุม Servo Motor ให้หมุนไปที่องศา 90 โดยใช้ myservo.write(90) และรอ 1 วินาที (delay(1000)) หลังจากนั้นแสดงข้อความ "CLOSING IN.." และนับถอยหลังากนั้นเริ่มทำการแกว่ง LED เพื่อสร้างเอฟเฟกต์



void close

ฟังก์ชันสำหรับปิดถัง

เรียก lcd.clear() เพื่อลบข้อมูลบนหน้าจอ LCD และ lcd.setCursor() และ lcd.print() เพื่อแสดงข้อความ "CLOSE" ทำการแกว่ง LED เพื่อสร้างเอฟเฟกต์ปิดถัง โดยสลับสี LED เขียวเปิด-ปิด ควบคุม Servo Motor ให้หมุนกลับไปที่องศา 0 โดยใช้ myservo.write(0) และรอ 1 วินาที. คืนข้อความหน้าจอ LCD กลับไปสู่เริ่มต้น "SMACK MY TRASH" และ "SMART BIN" คือ 30 cm. จะให้แสดงในจอ LCD แถวแรกว่า STATUS


int check_full

ฟังก์ชันเช็กถังขยะเต็ม

ในฟังก์ชัน check_full(), ตรวจสอบว่าถังขยะเต็มหรือไม่โดยใช้เซนเซอร์ Ultrasonic
ส่งสัญญาณ Ultrasonic เพื่อวัดระยะทางจากเซนเซอร์ Ultrasonic
ถ้าระยะทางที่วัดได้อยู่ในช่วง 100 ถึง 500 หน่วยไมครอนวิวเมตร,
ฟังก์ชันจะส่งค่า 1 ซึ่งหมายถึงถังขยะเต็ม ในกรณีอื่นๆ
ฟังก์ชันจะส่งค่า 0 ซึ่งหมายถึงถังขยะไม่เต็ม


void loop

ลูปการทำงาน

ในฟังก์ชัน loop(), โค้ดทำการทำงานตลอดเวลาโดยตรวจสอบสถานะและระยะทางจากเซนเซอร์ Ultrasonic
โค้ดเริ่มต้นโดยส่งสัญญาณ Ultrasonic เพื่อวัดระยะทางจากเซนเซอร์ Ultrasonic ด้านนอก
ตรวจสอบว่าถังหรือห้องน้ำเต็มหรือไม่โดยเรียกใช้ check_full()
ถ้าถังขยะเต็ม, เรียก full() เพื่อแสดงสถานะว่าถังเต็ม
หากระยะทางที่วัดได้อยู่ในช่วง 0 ถึง 10 หน่วยไมครอนวิวเมตร (หรือเข้าขอบเขตใกล้ถังขยะ),
เรียก open() เพื่อเปิดถังขยะ และตรวจสอบว่าถังขยะเต็มหรือไม่โดยเรียก check_full() อีกครั้ง.
ถ้าสถานะคือ 1, กำหนด LCD และ LED เพื่อแสดงสถานะเริ่มต้นและแสดงสถานะสีน้ำเงินและสีเขียวในโหมดที่ไม่มีการเปิดถังขยะ
